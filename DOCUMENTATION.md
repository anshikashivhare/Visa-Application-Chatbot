# Project Documentation

## Project Overview
A small static site + Node server for a "visa application guide" project. Contains an HTML front-end and a simple Node server.

## Prerequisites
- Node.js (v14+ recommended)
- npm

## Install
From the project root, run:

```bash
npm install
```

## Run (development)
- To run the Node server (if used):

```bash
node server.js
```

- Alternatively, open `index.html` directly in a browser for the static version.

## Files
- `index.html` — main frontend page
- `styles.css` — styles
- `script.js` — client-side JS
- `server.js` — Node server (optional)
- `package.json` — project metadata and scripts
- `assets/` — images and static assets

## Quick Start
1. Install dependencies: `npm install`
2. Start the server: `node server.js`
3. Open http://localhost:3000 (or the port shown by the server) in your browser, or open `index.html` directly.
